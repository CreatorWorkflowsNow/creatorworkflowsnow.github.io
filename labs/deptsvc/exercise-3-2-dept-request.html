Department Request Record Producer

Introduction‚Äã

Now that we have a data model created and our workspace experience ready to go, let's create some record producers so we can start getting some data into those tables. You can think of a record producer as an end user facing form to collect all necessary data to complete a request, address an issue, or answer an inquiry.

Let's start‚Äã

  1. Select the (+) Add link in the Experience Section Select the Record Producer experience Then select Begin on the confirmation screen

  1. Enter Dept Requests in the name field and provide a Short Description Select Continue

  1. Select Edit record producer

  1. Select Destination on the left. Set the Record submission table field to Dept Request

  1. Select the Location link on the left and then the Browse link\ Move Service Catalog from Available to Selected\ Select Save selections

  1. Select Browse in the Categories section\ Move Dept from Available to Selected\ Select Save selections

  1. Select the Questions link on the left\ Select Insert new question

  1. Set the Question type to Choice\ Set the Question subtype to Dropdown (fixed values)

  1. Check Map to a specific field on the table\ Choose Request type for the Table field\ Enter Request Type for the Question label\ Check Mandatory

    You may need to scroll down to see all fields

    Select Continue to Choices ->

  1. Check the Include none choice Check (+) to start adding Choices

  1. Enter the Display name for each of Request Type choice that you added to the Dept Request table earlier\ Select Insert Question when all choices have been added

  1. Select + Insert\ Select New Question

  1. Set the Question type to Text\ Set the Question subtype to Single-line

  1. Check Map to a specific field on the table\ Choose Short Description for the Table field\ Enter Request Summary for the Question label\ Check Mandatory

    You may need to scroll down to see all fields

    Select Insert Question

  1. Select + Insert\ Select New question

  1. Choose Text for the Question Type\ Choose Multi-line for the Question subtype

  1. Check Map to a specific field on the table\ Set the Table field to Description\ Set the Question label to Request Details

Select **Insert Question**
  1. Select the Review and Submit link on the left and then the Submit button on the right\ Select Return to my application
