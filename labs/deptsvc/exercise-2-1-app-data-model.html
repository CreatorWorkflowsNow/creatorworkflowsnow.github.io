Base Dept Task Table

Let's start‚Äã

  1. Once the App Home tab opens, select Add a table or upload a spreadsheet

  1. Select Create a table and then the Begin on the next screen

  1. Select the Create from an existing table option and then Continue

  1. Set the Table field to Task and select Continue

  1. Set the Table label field to Dept Task\ Select Make extensible\ Select Auto number\ Set the Prefix field to DEPTTASK

  1. The next step is to setup permission for the Dept Task table. Assign full permissions to the admin role by selecting All. Select the Create, Read and Write permissions for the user role

  1. Select Done once the table is created to return to App Home
